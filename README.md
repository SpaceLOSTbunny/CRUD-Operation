# 📚 Student Record Management System (CRUD)

A simple **Student Record Management System** built using **HTML, CSS, and JavaScript**. This project demonstrates the basic **CRUD (Create, Read, Update, Delete)** operations while managing student records and automatically calculating academic results.

## ✨ Features

* ➕ Add new student records
* 👨‍🎓 Store student name and marks for two subjects
* 🧮 Automatically calculate:

  * Total Marks
  * Percentage
* 📋 Display all student records in a table
* ✏️ Edit existing student details and marks
* 🗑️ Delete student records
* 🎨 Simple, responsive, and beginner-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🚀 How It Works

1. Enter the student's name.
2. Enter marks for Subject 1 and Subject 2.
3. Click **Add** to save the record.
4. The application automatically calculates the **Total Marks** and **Percentage**.
5. View all records in the table.
6. Use the **Edit** button to update a student's details or marks.
7. Use the **Delete** button to remove a student record.
